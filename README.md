# React Netflix Trends Dashboard - Final

This project includes:
- Netflix-themed dashboard (dark + red accents)
- Sidebar with tabs (KPIs / Map / Concentrado)
- Recharts charts
- React-Leaflet map using GeoJSON
- CSV loader (place netflix_titles.csv in public/)

Run:
1. npm install
2. npm run dev
3. Open http://localhost:5173
